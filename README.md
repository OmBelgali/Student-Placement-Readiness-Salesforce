# Student-Placement-Readiness-Salesforce
📌 Project Overview

This project is a Salesforce-based Student Placement Readiness Management System designed to automatically evaluate student readiness for campus placements based on skills, certifications, and academic performance (CGPA).

The system uses Salesforce declarative tools such as custom objects, record-triggered flows, reports, and dashboards to eliminate manual evaluation and provide real-time readiness insights.

🎯 Key Features

Automated skill score calculation using Salesforce Flows
Certification-based scoring and readiness evaluation
Real-time placement readiness classification (Ready / Almost Ready / Not Ready)
Visual insights through Salesforce reports and dashboards
Clicks Not Code implementation (no Apex required)

🛠 Technologies Used
Salesforce Developer Org
Salesforce Flows (Record-Triggered)
Custom Objects and Tabs
Reports and Dashboards
